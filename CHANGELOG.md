# Changelog

All notable changes to LoginChase are documented here.

## [1.0.0] - 2026-07-19

### Added

- Accessible fake login challenge with ten randomized evasive reactions.
- Guaranteed fifth-attempt prompt and tenth-attempt fake dashboard celebration.
- Mouse, touch, pen, and keyboard interaction support.
- Attempt tracking, local sound effects, sound toggle, reset, and replay controls.
- Responsive, high-contrast, and reduced-motion presentation.
- Enforced privacy architecture with no credential reads, persistence, or transmission.
- Static verification, production artifact generation, hosting security headers, and deployment documentation.
